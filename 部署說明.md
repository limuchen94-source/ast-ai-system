## 第一步：申請 Gemini API Key（免費）

1. 開啟瀏覽器，前往 **https://aistudio.google.com/app/apikey**
2. 用你的 Google 帳號登入
3. 點擊右上角 **「Create API key」** 藍色按鈕
4. 在彈出視窗中選擇 **「Create API key in new project」**
5. 複製產生的 API Key（格式類似：`AIzaSy...`），**請妥善保存，之後貼入平台側邊欄即可**

> **免費額度（2025年）：**
> - 每分鐘 15 次請求
> - 每天 1,500 次請求
> - 每月 100 萬 Token（足夠一般學生日常練習使用）

## 第二步：將程式碼上傳至 GitHub

### 方法 A：直接上傳（不需要懂 Git，推薦新手）

1. 前往 **https://github.com** 並登入
2. 點擊右上角 **「+」→「New repository」**
3. Repository name 填：`ast-ai-system`
4. 選擇 **Public**（Streamlit Cloud 免費版需要公開）
5. 勾選 **「Add a README file」**
6. 點擊 **「Create repository」**
7. 進入新建的 repository，點擊 **「Add file」→「Upload files」**
8. 將以下 **4 個檔案** 拖曳上傳：
   - `app.py`
   - `requirements.txt`
   - `.streamlit/config.toml`（需先建立 `.streamlit` 資料夾：點「Add file」→「Create new file」，檔名填 `.streamlit/config.toml`，貼上內容）

> **上傳 `.streamlit/config.toml` 的方法：**
> 在 GitHub 頁面點「Add file」→「Create new file」
> 在檔名欄位輸入：`.streamlit/config.toml`
> 將 config.toml 的內容貼上，點「Commit new file」

### 方法 B：使用 Git 命令（懂 Git 的用這個）
```bash
git init
git add .
git commit -m "AST AI System initial commit"
git branch -M main
git remote add origin https://github.com/你的帳號/ast-ai-system.git
git push -u origin main
```

## 第三步：部署到 Streamlit Cloud（免費永久網址）

1. 前往 **https://streamlit.io/cloud** 並點擊 **「Sign up」**
2. 選擇 **「Continue with GitHub」**，授權 Streamlit 存取你的 GitHub
3. 登入後點擊右上角 **「New app」**
4. 填寫以下資訊：
   - **Repository**：選擇 `你的帳號/ast-ai-system`
   - **Branch**：`main`
   - **Main file path**：`app.py`
5. 點擊 **「Deploy!」**
6. 等待約 **2-5 分鐘**，部署完成後會出現你的專屬網址

   格式類似：`https://ast-ai-system-xxxx.streamlit.app`

7. **分享這個網址**，任何人在任何裝置都可以開啟使用！

## 常見問題

**Q：API Key 安全嗎？**
A：API Key 只在你的瀏覽器 Session 中存在，不會被儲存到伺服器。但建議不要把 Key 分享給不信任的人。

**Q：免費額度用完了怎麼辦？**
A：Google AI Studio 的免費額度非常充裕（每天 1,500 次）。一般學生每天練習 20-30 題完全夠用。

**Q：手機可以用嗎？**
A：可以。Streamlit 自動支援響應式設計，手機開啟網址即可正常使用。

**Q：資料會保存嗎？**
A：目前為 MVP 版本，學習記錄保存在瀏覽器 Session 中，關閉分頁後清除。未來版本可加入 Supabase 資料庫實現長期保存。

**Q：程式碼可以修改嗎？**
A：當然可以。修改 GitHub 上的 `app.py` 後，Streamlit Cloud 會自動重新部署（約 1-2 分鐘）。
